# CSGClaw SVG Icons

These files mirror the icon components currently defined in `web/app/src/components/icons.tsx`.

Most icons use `currentColor` so they inherit color from CSS. `rooms.svg` and `room-plus.svg` also reference `var(--panel-soft)` for their circular background when used inside the app. The composer send icon is exported as `send.svg`.

Workspace sidebar icons extracted from Lucide live alongside the existing set as `workspace-*.svg` files, for example `workspace-messages.svg`, `workspace-agents.svg`, and `workspace-models.svg`.

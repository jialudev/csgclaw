## 本周发布更新

本周发布重点是补强团队任务协作、官方技能安装与 Agent 可视化，同时继续收紧 Windows、升级流程和 Codex 运行时稳定性。

### 更新内容

- 团队与 Agent 任务工作流进一步补齐，覆盖共享任务 API、CLI、HTTP 流程、实时更新和持久化存储，协作执行与状态跟踪更完整。
- Web UI 新增 Agent 活动面板，可查看 OpenClaw 运行时的 Agent 事件、工作状态，并支持从聊天直接进入 Agent 详情，排查与协作入口更直观。
- 现已支持从官方 SkillHub 安装远程技能，并补上更安全的归档处理、分页和无效技能过滤等能力。
- Agent 创建与运行时处理进一步统一，包括模板、API 和资料页中的默认值记忆与运行时归一化，创建配置更一致。
- 前端将 Hub 重命名为 Resources，并同步更新徽标、图标和导航资源，资源入口与产品命名保持一致。
- Windows 上的 OpenClaw/Codex 兼容性继续提升，包括工作区隔离、审批文件挂载调整和任务状态写入修正，本地运行更稳定。
- 升级与运行时可靠性继续增强，包括更清晰的升级失败诊断、统一的 runtime config 兼容处理，以及删除 Agent 时更可靠的 Codex 运行时清理。
- Codex 相关集成继续补强，包括 Agent 模型现已支持 OpenCSG provider，以及工具调用和最终回复可接入 Agent 活动面板，相关使用反馈更完整。
- 多项 Web UI 稳定性问题已修复，包括模型 Provider 页面滚动、上下文错误跳转、环境编辑草稿崩溃，以及模板模式和详情页布局细节。

### 涉及版本

- `v0.3.15`: 扩展团队与 Agent 任务协作流程，支持官方远程技能安装，并修复 Windows OpenClaw 兼容性和多项 Web UI 稳定性问题。
- `v0.3.16`: 新增 Agent 活动可视化，统一 Agent 创建与 runtime 处理，并继续补强升级、Windows、Codex 和技能集成可靠性。

## Weekly Release Update

This week’s releases focused on stronger team task workflows, official skill distribution, and better agent visibility, while continuing to harden Windows behavior, upgrade flows, and Codex runtime reliability.

### What's Changed

- Team and agent task workflows now cover shared task APIs, CLI flows, HTTP handling, realtime updates, and persistent storage more completely, making collaborative execution and status tracking more reliable.
- The Web UI now includes an agent activity dashboard for OpenClaw runtime events and working status, with direct chat entry points into agent details.
- Official remote skills can now be installed from SkillHub, with safer archive handling, pagination, and invalid-skill filtering.
- Agent creation and runtime handling are more consistent across templates, APIs, and profile views, with remembered defaults and clearer runtime normalization.
- The frontend Hub surface has been renamed to Resources, with matching badge, icon, and navigation updates to keep the product surface consistent.
- OpenClaw and Codex flows on Windows were hardened with better workspace isolation, adjusted approval-file mounting, and safer task-state writes.
- Upgrade and runtime reliability improved with clearer upgrade failure diagnostics, unified runtime config compatibility handling, and more reliable Codex runtime cleanup when deleting agents.
- Codex integration was strengthened with OpenCSG provider support for agent models and by surfacing tool calls and final replies in the agent activity dashboard.
- Several Web UI stability issues were fixed, including model provider scrolling, contextual error routing, env editor draft crashes, and layout or edit-flow issues in template and detail views.

### Covered releases

- `v0.3.15`: Expanded team and agent task collaboration, added official remote skill installs, and fixed Windows OpenClaw compatibility plus several Web UI stability issues.
- `v0.3.16`: Added agent activity visibility, streamlined agent creation and runtime handling, and further improved upgrade, Windows, Codex, and skill integration reliability.
